Project Description

This project detects fraudulent banking transactions using Machine Learning. A Random Forest classifier is trained on transaction data to identify whether a transaction is genuine or fraudulent.

Technologies Used
Python
Pandas
Scikit-learn
Matplotlib
Dataset
Credit Card Fraud Detection Dataset
Contains transaction details and fraud labels.
Features
Data loading and preprocessing
Fraud detection using Random Forest
Accuracy evaluation
Visualization of fraud vs genuine transactions
How to Run

Install libraries:

pip install pandas scikit-learn matplotlib
Download the dataset and save it as creditcard.csv.

Run:

python fraud_detection.py
Output
Dataset preview
Fraud prediction results
Accuracy score
Classification report
Fraud vs Genuine transaction graph
Conclusion

This project uses machine learning to identify fraudulent banking transactions and helps improve banking security by detecting suspicious activities.
